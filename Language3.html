Python is typed dynamically, and obtained from garbage. It supports multiple programming paradigms, including structured programming (especially procedural), object-oriented, and functional. Because of its extensive standard library, Python is sometimes defined as "batteries included" language. Python was conceived of as a descendant to the ABC language in the late 1980s. Python 2.0, released in 2000, introduced features such as list comprehensions and a method of garbage collection capable of collecting reference cycles. Python 3.0, released in 2008, was a major language revision that is not fully backward-compatible, and most of the Python 2 code does not run unchanged on Python 3.
